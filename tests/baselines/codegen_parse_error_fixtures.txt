# Codegen intentional-skip baseline — fixtures the codegen round-trip
# verifier must skip because the source intentionally fails to parse
# cleanly. Codegen refuses to emit when parse produced errors, so those
# fixtures can never round-trip; the gate treats them as expected skips
# instead of either silently bumping the skip count or flagging them as
# regressions.
#
# Behavior:
#   - A fixture listed here that still triggers parse_errors_in_source
#     during round-trip is counted as `intentional-skip` and passes.
#   - A fixture listed here that now PARSES CLEANLY is an IMPROVEMENT
#     (the verifier prints a hint; remove it from this file).
#   - A fixture NOT listed here that triggers parse_errors_in_source is
#     counted as plain `skip` and surfaced under SHOW_SKIPS=1.
#
# Path is RELATIVE to tests/fixtures/.
